# Brand asset provenance

No official Automata Intelligence logo or wordmark is committed in this directory because a sufficiently crisp, unambiguous official asset could not be retrieved from the accessible official website during production. Ambiguous third-party search results were rejected.

The campaign uses clearly typeset company-name identification and does not represent that text treatment as an official wordmark. Add an official source asset here only after provenance, quality, and identity are verified.
